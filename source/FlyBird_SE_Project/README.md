# FlyBird_SE_Project 源码

这是 `FlyBird SE Project` 的源码目录，适用于基于 Qt 的继续开发与维护。

## 主要文件

- `main.cpp`：程序入口
- `game.cpp` / `game.h`：游戏主逻辑
- `bird.cpp` / `bird.h`：小鸟逻辑
- `pipe.cpp` / `pipe.h`：管道逻辑
- `flyBird.pro`：Qt qmake 工程文件
- `qrc.qrc`：资源文件配置
- `assets/images/`：图片资源

## 构建方式

推荐直接使用 Qt Creator 打开 `flyBird.pro`，选择 Qt 6 环境后进行编译和运行。
